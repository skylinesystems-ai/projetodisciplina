# Disciplina Suprema

Sistema RPG de disciplina diaria para treino corporal, XP, sequencia, Escudo de Honra, missoes, conquistas, progressao ate SUPREMO e temporadas lendarias.

## Rodar localmente

```bash
npm install
npm run dev
```

Acesse:

```text
http://127.0.0.1:5173/
```

Tambem existe uma entrada direta em:

```text
http://127.0.0.1:5173/supremo.html
```

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run test
npm run preview
```

## Recursos

- Pontuacao iniciando zerada.
- 15 niveis: Recruta ate SUPREMO.
- XP por flexoes, agachamentos e abdominais.
- Missoes diarias, semanais e epicas.
- Sequencia com bonus por 1, 5, 10, 30 e 100 dias.
- Escudo de Honra para proteger a sequencia.
- Sistema anti-abandono com missao de recuperacao.
- Badges, recompensas, ranking pessoal e Modo Lendario.
- Ao chegar em SUPREMO: aura, Hall da Fama, multiplicador permanente, insignia unica e certificado.
